Product Sales Reports

Given a time window, displays the sales by item from the order history.

Directions:

1. Input a start date (mm/dd/yyyy).
2. Input a end date (mm/dd/yyyy).
3. Click submit.


Most Sold Products Pairs

Given a time window, displays pairs of menu items that sell together often, sorted by most frequent.

Directions:

1. Input a start date (mm/dd/yyyy).
2. Input a end date (mm/dd/yyyy).
3. Click submit.


Restock Report

Displays the list of items whose current inventory is less than the item's minimum amount to have around before needing to restock.

Excess Report

Given a timestamp, displays the list of items that only sold less than 10% of their inventory between the timestamp and the current time, assuming no restocks have happened during the window.

Directions:

1. Input a start date (mm/dd/yyyy).
2. Click submit.
