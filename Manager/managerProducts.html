Add New Product

Directions:

1. Input new product's name.
2. Input new product's group type ("food") or ("drinks").
3. Input new product's subgroup type ("sandwiches", "cookie", "muffin") or ("coffee", "teas").
4. Input new product's price (XX.XX)
5. Select ingredients required in new product.
6. Click sumbit.


Update Product

Directions:

1. Input existing product's name.
2. Input existing product's updated price.
3. Select updated ingredients required in existing product.
4. Click sumbit.
